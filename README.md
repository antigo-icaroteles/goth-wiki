# GAME OF THRONES WIKI  

> Projeto Concluído (versão 1.0) 

## Descrição  

- Uma enciclopédia sobre os personagens de Game Of Thrones  

## Tecnologias Utilizadas  

- React + Next.JS  
- Axios  
- API: Game Of Thrones API (https://thronesapi.com/)  
- CSS  

## Para rodar o projeto  

``` 
  npm start
````
